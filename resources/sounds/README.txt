Thư mục này chứa các file âm thanh cho trò chơi cờ tướng.

Các file âm thanh cần thiết:
1. move.wav - Phát khi di chuyển quân cờ
2. capture.wav - Phát khi bắt quân cờ đối phương
3. check.wav - Phát khi chiếu tướng

Bạn có thể tải các file âm thanh từ các nguồn miễn phí và đặt vào thư mục này, hoặc tạo các file âm thanh của riêng bạn.

Nếu không có file âm thanh, ứng dụng sẽ vẫn hoạt động nhưng không phát ra âm thanh.
